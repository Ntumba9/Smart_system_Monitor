Hybrid Architecture

Purpose-built components for maximum performance and clarity

C

C Agent

Low-Level Collection

Efficient background daemon collecting raw system metrics via direct kernel interfaces with minimal overhead.

  • Direct syscall interfaces for maximum performance
  • Shared memory IPC for zero-copy data exchange
  • Sub-1% CPU overhead even under heavy load
  • Real-time metric collection at 10Hz frequency
Py

Python Layer

Automation & Logic

Intelligent processing layer that analyzes metrics, applies rules, and triggers automated system responses.

  • Rule engine with temporal condition evaluation
  • Historical data storage in SQLite database
  • Plugin system for custom automation actions
  • WebSocket server for real-time updates
Jv

Java Controller

Service Management

Central service manager providing REST API, authentication, and orchestration between system components.

  • Spring Boot REST API with OpenAPI documentation
  • JWT-based authentication and rate limiting
  • Service lifecycle management (start/stop/monitor)
  • Configuration management and validation
JS

JavaScript Dashboard

Visualization & Control

Elegant, responsive web interface providing real-time visualization and control over the entire monitoring system.

  • Real-time charts with smooth animations
  • WebSocket integration for live updates
  • Rule configuration and management interface
  • Progressive Web App capabilities
